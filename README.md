## lavalink-server
### Lavalink Server for Linux and Windows
<br>

1. Clone the repository or download as zip

```bash
git clone https://github.com/kniferftw/lavalink-server.git
```

2. Navigate into dir 

```bash
cd lavalink-server
```

3. Install packages

```bash
npm install
```

4. Run

```bash
# For Windows
npm i win

# For Linux
npm i linux
```
